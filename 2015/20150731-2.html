分支开发

在github客户端操作
创建新分支：项目文件名 -> Manage -> 右侧Unpbulish -> Branch 建立分支名为BUG
在新分支中：修改项目文件内容 -> Commit to BUG 提交修改内容到BUG分支
合并分支内容：项目文件名 -> Manage -> BUG 移动到左侧Merge ，master移动到右侧Merge -> 点击Merge
